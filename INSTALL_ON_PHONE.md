# 📱 내 폰에 설치하기 가이드

Google Play Store 없이 직접 내 폰에 2048 게임을 설치하는 방법입니다.

## 🚀 방법 1: 웹 서버로 바로 사용 (가장 빠름! ⭐)

APK 없이도 앱처럼 사용 가능합니다.

### 준비물:
- 컴퓨터와 핸드폰이 **같은 Wi-Fi**에 연결되어 있어야 함
- Python 3 설치 (대부분의 컴퓨터에 기본 설치됨)

### 단계:

#### 1️⃣ 서버 시작

**Windows:**
```cmd
cd "C:\Users\student\Desktop\Vibe coding\Day1\module_2"
python simple_server.py
```

**또는 Python이 없다면:**
```cmd
cd "C:\Users\student\Desktop\Vibe coding\Day1\module_2"
python -m http.server 8000
```

#### 2️⃣ IP 주소 확인

서버가 시작되면 다음과 같은 메시지가 표시됩니다:
```
📱 핸드폰에서 접속하기:
   http://192.168.x.x:8000
```

**IP 주소를 모를 때:**

Windows:
```cmd
ipconfig
```
→ "무선 LAN 어댑터 Wi-Fi"의 "IPv4 주소" 확인

#### 3️⃣ 핸드폰에서 접속

1. 핸드폰 Chrome 브라우저 열기
2. 주소창에 입력: `http://컴퓨터IP:8000`
   - 예: `http://192.168.0.100:8000`
3. 게임이 바로 실행됩니다!

#### 4️⃣ 홈 화면에 추가 (앱처럼 사용)

1. Chrome 메뉴 (⋮) > "홈 화면에 추가"
2. "추가" 탭
3. 이제 일반 앱처럼 사용 가능!

### 장점:
- ✅ APK 빌드 불필요
- ✅ 즉시 사용 가능
- ✅ 업데이트 쉬움 (파일만 수정하면 됨)

### 단점:
- ❌ 컴퓨터 서버가 켜져 있어야 함
- ❌ 같은 Wi-Fi 연결 필요

## 🎯 방법 2: APK 직접 설치 (완전 독립 실행)

컴퓨터 없이도 폰에서 독립적으로 실행됩니다.

### 옵션 A: Android Studio로 빌드 (20분)

#### 준비:
1. Android Studio 설치
   - https://developer.android.com/studio
   - 약 2-3GB 다운로드

#### 빌드 단계:

1. **Android Studio 실행**
2. **프로젝트 열기**
   ```
   Open > android 폴더 선택
   ```

3. **Gradle 동기화 대기** (자동)
   - 첫 실행 시 5-10분 소요
   - 인터넷 연결 필요

4. **APK 빌드**
   ```
   Build > Build Bundle(s) / APK(s) > Build APK(s)
   ```

5. **APK 찾기**
   ```
   android/app/build/outputs/apk/debug/app-debug.apk
   ```

6. **핸드폰으로 전송**
   - 카카오톡, 이메일, USB 등으로 전송

7. **핸드폰에서 설치**
   - 파일 관리자에서 APK 파일 탭
   - "설치" 탭
   - "출처를 알 수 없는 앱 설치 허용" 필요할 수 있음

### 옵션 B: 명령줄로 빌드 (Gradle 사용)

Android Studio 없이 명령줄로도 빌드 가능합니다.

#### 준비:
- JDK 17 이상 설치
- Android SDK 설치

#### 명령어:

**Windows:**
```cmd
cd android
gradlew assembleDebug
```

**Mac/Linux:**
```bash
cd android
./gradlew assembleDebug
```

빌드된 APK 위치:
```
android/app/build/outputs/apk/debug/app-debug.apk
```

## 📦 방법 3: 온라인 빌드 서비스 (중급)

웹사이트에서 APK를 빌드해주는 서비스입니다.

### 사이트:
- **AppGyver** (무료)
- **Appy Pie** (무료 체험)
- **BuildFire** (무료 플랜)

### 단계:
1. 사이트 가입
2. 새 프로젝트 생성
3. 웹 파일 업로드 (index.html, css, js)
4. 빌드 버튼 클릭
5. APK 다운로드

### 단점:
- 가입 필요
- 업로드 시간 소요
- 일부 기능 제한

## 🔧 APK 설치 시 문제 해결

### "출처를 알 수 없는 앱" 오류

**Android 8.0 이상:**
1. 설정 > 보안
2. "출처를 알 수 없는 앱 설치" 또는 "알 수 없는 소스"
3. 사용하는 파일 관리자/브라우저 선택
4. "허용" 켜기

**Android 7.0 이하:**
1. 설정 > 보안
2. "알 수 없는 소스" 허용

### "앱이 설치되지 않았습니다" 오류

**해결책:**
1. 기존에 같은 패키지명의 앱이 있다면 삭제
2. 저장 공간 확인
3. APK 파일 다시 다운로드

### 앱이 실행되지 않음

**확인 사항:**
1. Android 7.0 (API 24) 이상인지 확인
2. 저장소 권한 허용
3. 앱 삭제 후 재설치
